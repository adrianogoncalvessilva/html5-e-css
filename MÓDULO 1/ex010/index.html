Usando links externos

Você pode acessar o meu repositório público no GitHub

Você também pode acessar o meu canal gratuito no YouTub

Quando o link é para um site fora do seu, é recomendado que se use as tags " target="_blank" e " rel="external" - para que o site seja aberto em uma nova aba. Caso contrário, o usuário abandonará o seu site

Links internos

Esta é a primeira página do site. Se você quiser, pode acessar também a minha segunda página.

O rel="next" é recomendado para ajudar nos mecanismos de busca do google. Indica que leva a próxima página do site

Você também pode acessar a nossa página de notícias

A tag "target="_self" serve para que a o link abra na mesma página. Ficar abrindo todos os links em abas diferentes não é legal.
